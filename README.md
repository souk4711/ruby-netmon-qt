# NetmonQt

A simple network monitor on Linux.

## Requirements

- [Ruby](https://www.ruby-lang.org/)
- [Qt](https://www.qt.io/)
- [LIBpcap](https://github.com/the-tcpdump-group/libpcap)

## Installation

```console
$ gem install ruby-netmon-qt
```

## Usage

To launch the GUI:

```console
$ netmon-qt
```
